Class Embed

A message embed; specifically, a rich embed. When setting properties of the embed, this class automatically checks their length against Discord's maximum lengths for embed properties.

example
new Embed()
    .setTitle(`A title`)
    .setDescription(`Some description`)
    .setColor(0x5865F2);
